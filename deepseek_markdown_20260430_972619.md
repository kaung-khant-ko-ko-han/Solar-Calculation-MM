ဤအစီရင်ခံစာအား အမေရိကန်ပြည်ထောင်စု အစိုးရအေဂျင်စီတစ်ခု၏ ပံ့ပိုးမှုဖြင့် ပြုလုပ်ခဲ့ပါသည်။ အမေရိကန်အစိုးရ၊ အေဂျင်စီများ သို့မဟုတ် ၎င်းတို့၏ ဝန်ထမ်းများမှ ဖော်ပြထားသော သတင်းအချက်အလက်များ၏ တိကျမှု၊ ပြည့်စုံမှု၊ အသုံးဝင်မှုတို့အတွက် မည်သည့်အာမခံချက်မျှ (အတိအလင်း သို့မဟုတ် သွယ်ဝိုက်၍) မပေးသည့်အပြင် မည်သည့်တရားဝင်တာဝန်ယူမှုမျှလည်း မရှိပါ။ ထို့အပြင် ၎င်းကိုအသုံးပြုခြင်းသည် ပုဂ္ဂလိကဉာဏပစ္စည်းမူပိုင်ခွင့်များကို ချိုးဖောက်မည်မဟုတ်ကြောင်း ကိုယ်စားပြုခြင်းလည်းမရှိပါ။ ဤတွင် ကုန်သွယ်မှုအမည်၊ ကုန်အမှတ်တံဆိပ်၊ ထုတ်လုပ်သူ သို့မဟုတ် အခြားနည်းလမ်းဖြင့် သီးခြားကူးသန်းရောင်းဝယ်ရေးဆိုင်ရာ ထုတ်ကုန်၊ လုပ်ငန်းစဉ်၊ သို့မဟုတ် ဝန်ဆောင်မှုကို ကိုးကားထားခြင်းသည် အမေရိကန်အစိုးရ သို့မဟုတ် ၎င်း၏အေဂျင်စီမှ ထောက်ခံအားပေးခြင်း၊ အကြံပြုခြင်း သို့မဟုတ် နှစ်သက်သဘောကျခြင်းကို ဆိုလိုခြင်းမဟုတ်ပါ။ ဤတွင် ဖော်ပြထားသော စာရေးသူ၏ အမြင်များနှင့် သဘောထားများသည် အမေရိကန်အစိုးရ သို့မဟုတ် ၎င်း၏အေဂျင်စီ၏ အမြင်ကို အတည်ပြုဖော်ပြခြင်း သို့မဟုတ် ထင်ဟပ်ခြင်းလည်းမဟုတ်ပါ။

## ၁ နိဒါန်းနှင့်သမိုင်းကြောင်း (Introduction and History)

PVWatts သည် Grid-ချိတ်ဆက်ထားသော photovoltaic (PV) စနစ်များ၏ စွမ်းအင်ထုတ်လုပ်မှုကို ခန့်မှန်းတွက်ချက်ပေးသည့် နာမည်ကြီး web application တစ်ခုဖြစ်သည်။ ၎င်းကို ကျွမ်းကျင်သူမဟုတ်သူများနှင့် အဆင့်မြင့်အသုံးပြုသူများအတွက်ပါ ရိုးရှင်းလွယ်ကူစွာ အသုံးပြုနိုင်ရန် ဒီဇိုင်းထုတ်ထားသည်။ PVWatts သည် စနစ်၏အမျိုးအစား၊ ဖွဲ့စည်းပုံနှင့် လည်ပတ်ပုံတို့နှင့်ပတ်သက်၍ ခန့်မှန်းချက်များစွာပြုလုပ်ထားခြင်းဖြင့် PV စနစ်များကို တိကျစွာပုံစံတူတွက်ချက်ခြင်း၏ ရှုပ်ထွေးမှုများကို အသုံးပြုသူမသိစေရန် ကွယ်ဝှက်ထားသည်။ သို့ဖြစ်ရာ ရလဒ်များသည် ပုံမှန်ရာသီဥတုရှိသော နှစ်တစ်နှစ်တွင် လည်ပတ်နေသည့် အလားတူစနစ်အတွက် ကိုယ်စားပြုခန့်မှန်းချက်အဖြစ် အဓိပ္ပါယ်ကောက်ယူသင့်သည်။ ရေရှည်သမိုင်းဝင် ပုံမှန်အခြေအနေများကို ကိုယ်စားပြုသော ရာသီဥတုဒေတာအတွက် နှစ်စဉ်စွမ်းအင်ပမာဏတွင် အမှားသည် ±10% အထိ၊ လစဉ်ပမာဏတွင် ±30% အထိရှိနိုင်သည်။ သတ်မှတ်ထားသော နှစ်တစ်နှစ်အတွက် အမှန်တကယ်စွမ်းဆောင်ရည်သည် ရေရှည်ပျမ်းမျှမှ နှစ်စဉ်အတွက် ±20% အထိ၊ လစဉ်တန်ဖိုးများအတွက် ±40% အထိ ကွဲလွဲနိုင်သည်။

PVWatts ကို ၁၉၉၉ ခုနှစ်မှစ၍ အွန်လိုင်းတွင် ရရှိခဲ့ပြီး ဗားရှင်း ၁ တွင် မူလ algorithms များသည် ၁၉၈၀ ပြည့်လွန်နှစ်များတွင် တီထွင်ခဲ့သော Sandia PVFORM tool ၏ နည်းလမ်းပေါ်တွင် အများအားဖြင့်အခြေခံခဲ့သည်။ PVWatts ဗားရှင်း ၁ အတွက် နည်းပညာဆိုင်ရာကိုးကားစာအုပ်ကို [1] တွင်ရရှိနိုင်သည်။ ထိုအချိန်မှစ၍ PVWatts ၏ ဗားရှင်းအများအပြားကို ထုတ်လုပ်ခဲ့သော်လည်း စနစ်စွမ်းဆောင်ရည်တွက်ချက်မှုများမှာ ဗားရှင်း ၁ နှင့် အကြမ်းဖျင်းတူညီခဲ့သည်။ ဗားရှင်းသမိုင်းကို ဇယား ၁ တွင်ပြသထားသည်။

၂၀၁၃ ခုနှစ်တွင် NREL မှ PVWatts အွန်လိုင်း web application အား အမြင်အာရုံဆွဲဆောင်မှုနှင့် လုပ်ဆောင်ချက်များကို အဆင့်မြှင့်တင်ရန်၊ ဆက်လက်ထိန်းသိမ်းမှုဝန်ထုပ်ဝန်ပိုးကို လျှော့ချရန် ဗားရှင်းများကို ပေါင်းစည်းရန်၊ ခေတ်မီ photovoltaic စနစ်များ၏ အမှန်တကယ်စွမ်းဆောင်ရည်နှင့် ကိုက်ညီစေရန် စွမ်းအင်ခန့်မှန်းခြင်း algorithms များကို အဆင့်မြှင့်တင်ရန် ပြန်လည်မွမ်းမံခြင်းလုပ်ငန်းစဉ်ကို စတင်ခဲ့သည်။

PVWatts သည် photovoltaic စနစ်မှ စွမ်းအင်ထုတ်လုပ်မှုကို အမြန်ခန့်မှန်းချက်ရယူရန် အသုံးဝင်သောကိရိယာတစ်ခုဖြစ်သော်လည်း ပိုမိုတိကျသောခန့်မှန်းချက်များအတွက် အခြားသော ပိုမိုခေတ်မီဆန်းပြားသော ကိရိယာများစွာလည်း ရရှိနိုင်သည်။ System Advisor Model (SAM) သည် NREL မှတီထွင်ထားသော အခမဲ့ desktop application ဖြစ်ပြီး အသုံးပြုသူများအား PV စနစ်များကို ပိုမိုအသေးစိတ်ကျစွာ ပုံစံတူတွက်ချက်ခွင့်ပေးသည် [4]။ SAM တွင် လူနေအိမ်သုံး၊ စီးပွားရေးလုပ်ငန်းသုံးနှင့် အသုံးဝင်မှုအတိုင်းအတာစနစ်များအတွက် အသေးစိတ်စီးပွားရေးခွဲခြမ်းစိတ်ဖြာမှုများအပြင် အာရုံစူးစိုက်နေရောင်ခြည်စွမ်းအင် (CSP)၊ လေစွမ်းအင်၊ နေရောင်ခြည်ရေအပူပေးစနစ် နှင့် ဘူမိအပူစနစ်များအတွက် စွမ်းဆောင်ရည်ပုံစံတူတွက်ချက်မှုများလည်းပါဝင်သည်။

PVsyst သည် အသေးစိတ် PV စွမ်းဆောင်ရည်ပုံစံတူတွက်ချက်မှုအတွက် နောက်ထပ်ရွေးချယ်စရာတစ်ခုဖြစ်ပြီး စီးပွားဖြစ်ဆော့ဖ်ဝဲထုတ်ကုန်တစ်ခုဖြစ်သည် [5]။

## ၂ ဗားရှင်း ၅ တွင် ပြောင်းလဲမှုများ ခြုံငုံသုံးသပ်ချက် (Overview of Changes in Version 5)

PVWatts ဗားရှင်း ၅ သည် ပုံသေသတ်မှတ်တန်ဖိုးများနှင့် algorithms နှစ်ခုစလုံးတွင် ဘက်စုံအဆင့်မြှင့်တင်မှုတစ်ခုဖြစ်သည်။ ပြောင်းလဲမှုများသည် input အရေအတွက်ကို အနည်းဆုံးထားရှိရင်း ပုံမှန်စနစ်များကို ပိုမိုတိကျစွာပုံစံတူတွက်ချက်နိုင်ရန်နှင့် အသုံးပြုသူအပေါ် အလွန်အကျွံ နည်းပညာဝန်ထုပ်ဝန်ပိုးမဖြစ်စေဘဲ အသုံးများသော နည်းပညာရွေးချယ်စရာများကြား ခွဲခြားနိုင်သည့်နည်းလမ်းကိုပေးရန် ဒီဇိုင်းထုတ်ထားသည်။

ယခင်ဗားရှင်းများ၏ PVWatts တွက်ချက်မှုများကို [1] တွင်ဖော်ပြထားသည်။ PVWatts V5 တွင် အဓိကပြောင်းလဲမှုများမှာ-

1.  "Standard", "Premium", သို့မဟုတ် "Thin film" module အမျိုးအစားကို ရွေးချယ်နိုင်ခြင်း။
2.  DC-to-AC nameplate sizing ratio ကို သတ်မှတ်နိုင်ခြင်း။
3.  Module model တွင် အလင်းရောင်နည်းပါးချိန်၌ quadratic correction မပါဝင်တော့ခြင်း။
4.  စနစ်ဆုံးရှုံးမှုများကို ရာခိုင်နှုန်းဖြင့် သတ်မှတ်ပြီး ပုံသေတန်ဖိုးမှာ 14% ဖြစ်သည်။ ၎င်းသည် PVWatts V1 ရှိ ယခင် DC-to-AC derate factor ကို အစားထိုးသည်။
5.  Inverter စွမ်းဆောင်ရည်မျဉ်းကွေးကို ၂၀၁၀ ခုနှစ်နောက်ပိုင်း ထုတ်လုပ်သော inverter များ၏ ဒေတာများကို ကိန်းဂဏန်းဆိုင်ရာ ခွဲခြမ်းစိတ်ဖြာမှုမှ ရယူထားသည်။ အသုံးပြုသူမှ nominal inverter စွမ်းဆောင်ရည်ကို ထည့်သွင်းနိုင်သည်။
6.  ဝင်ရိုးတစ်လုံးတည်းလှည့်သော (One-axis tracking) စနစ်များသည် အတန်းကြားအကွာအဝေးပေါ်မူတည်၍ linear beam+diffuse အလိုအလျောက်အရိပ်ကျခြင်းဆုံးရှုံးမှုကို ခန့်မှန်းတွက်ချက်နိုင်သည် သို့မဟုတ် backtracking ကိုအသုံးပြုသည်။
7.  Albedo (မြေပြင်မှအလင်းပြန်နှုန်း) ကို TMY3, EPW, သို့မဟုတ် SAM/CSV ရာသီဥတုဖိုင်တွင် နာရီတိုင်းအတွက် အတိအလင်းသတ်မှတ်မထားပါက 0.2 ဟု ပုံသေသတ်မှတ်ထားသည်။

## ၃ မော်ဒယ်၏ ထည့်သွင်းအချက်အလက်များ (Model Inputs)

PVWatts သည် ဇယား ၂ တွင်ဖော်ပြထားသည့်အတိုင်း အနည်းဆုံး PV စနစ်အသေးစိတ်အချက်အလက်များကို လိုအပ်သည်။ Module temperature coefficients ကဲ့သို့သော အခြားစွမ်းဆောင်ရည်ဆိုင်ရာ parameters များကို အသုံးပြုသူမမြင်ရအောင် ကွယ်ဝှက်ထားသည်။

**ဇယား ၂။ ထည့်သွင်းအချက်အလက်များ (Input Parameters)**

| ကွက်လပ် | ယူနစ် | ပုံသေတန်ဖိုး |
| :--- | :--- | :--- |
| စနစ်အရွယ်အစား (System size) | kW (DC) | 4 |
| Module အမျိုးအစား (Module type) | Standard, Premium, Thin film | Standard |
| စနစ်ဆုံးရှုံးမှုများ (System losses) | % | 14 |
| Array အမျိုးအစား (Array type) | Fixed open rack, Fixed roof mount, 1-Axis, Backtracked 1-Axis, 2-Axis | Fixed open rack |
| လျှောစောက်ထောင့် (Tilt angle) | ဒီဂရီ | Site Latitude (ဆိုဒ်၏လတ္တီတွဒ်) |
| အရပ်မျက်နှာထောင့် (Azimuth angle) | ဒီဂရီ | မြောက်ကမ္ဘာခြမ်းတွင် ၁၈၀°, တောင်ကမ္ဘာခြမ်းတွင် 0° |
| **အဆင့်မြင့် ထည့်သွင်းအချက်အလက်များ (Advanced inputs)** | | |
| DC/AC အချိုး (DC/AC ratio) | အချိုး | 1.1 |
| Inverter စွမ်းဆောင်ရည် (Inverter efficiency) | % | 96 |
| GCR (1 Axis အတွက်သာ) (GCR) | အပိုင်းကိန်း | 0.4 |

Module အမျိုးအစား input အသစ်သည် မတူညီသောနည်းပညာရွေးချယ်စရာများကြား ခွဲခြားနိုင်ရန် ကူညီပေးသည်။ "Standard" ရွေးချယ်မှုသည် စွမ်းဆောင်ရည် 14-17% အတွင်းရှိသော ပုံမှန် poly- သို့မဟုတ် mono-crystalline silicon module များကို ကိုယ်စားပြုသည်။ "Premium" ရွေးချယ်မှုသည် anti-reflective coatings ပါရှိပြီး temperature coefficients နိမ့်သော စွမ်းဆောင်ရည်မြင့် (∼18-20%) monocrystalline silicon module များကို ပုံစံတူပြုလုပ်ရန် သင့်လျော်သည်။ Thin film ရွေးချယ်မှုသည် စွမ်းဆောင်ရည်နိမ့်ခြင်း (∼11%) နှင့် ၂၀၁၃ ခုနှစ်အထိ တပ်ဆင်ထားသော thin film module အများစုကို ကိုယ်စားပြုသည့် သိသိသာသာနိမ့်သော temperature coefficient ကို ယူဆသည်။ အသုံးနည်းသော thin film module နည်းပညာများသည် ပုံသေသတ်မှတ်ချက်နှင့် အလွန်ကွာခြားသော temperature coefficients ရှိနိုင်ကြောင်း သတိပြုရန် အရေးကြီးသည်။ Module ဒေတာစာရွက်ပေါ်ရှိ အချက်အလက်များကိုအခြေခံ၍ စဉ်းစားဆင်ခြင်နေသော module အမျိုးအစားအတွက် အသင့်တော်ဆုံးရွေးချယ်မှုကို ဆုံးဖြတ်ရန်မှာ အသုံးပြုသူ၏တာဝန်ဖြစ်သည်။ "Standard" ရွေးချယ်မှုသည် ပုံမှန်ပဏာမ ခွဲခြမ်းစိတ်ဖြာမှုများအတွက် သင့်လျော်ပြီး PVWatts V1 နှင့် အလားတူဆုံးဖြစ်သည်။

အဆင့်မြင့် input အသစ်များက အသုံးပြုသူအား DC-to-AC sizing ratio နှင့် ground coverage ratio တို့ကို သတ်မှတ်ခွင့်ပြုထားပြီး ၎င်းမှာ PVWatts ၏မူလဗားရှင်းများတွင် မဖြစ်နိုင်ခဲ့ပါ။ PVWatts V1 သည် စနစ်၏ DC nameplate ပမာဏကို AC nameplate ပမာဏနှင့် တူညီသည်ဟု ယူဆခဲ့သည်။ ခေတ်မီစနစ်များတွင် photovoltaic array ကို ၎င်း၏ DC rating သည် inverter ၏ AC rating ထက် ပိုမိုမြင့်မားအောင် မကြာခဏ အရွယ်အစားသတ်မှတ်လေ့ရှိသည်။ တည်နေရာပေါ်မူတည်၍ ၎င်းမှာ နေအလင်းရောင်အမြင့်ဆုံးနာရီများတွင် output ကို ကန့်သတ်ပိတ်ပစ်ခြင်း (clipping) ဖြစ်နိုင်သော်လည်း နေ့စ၏အစနှင့်အဆုံးပိုင်းများတွင် စွမ်းအင်ပိုမိုဖမ်းယူရန် လိုလားဖွယ်ဖြစ်နိုင်သည်။ DC-to-AC ratio input အသစ်၏ ပုံသေတန်ဖိုးမှာ 1.1 ဖြစ်သည်။ 4 kW DC စနစ်အတွက် ၎င်းသည် 3.63 kW AC inverter rating ကိုဖြစ်ပေါ်စေလိမ့်မည်။

Ground coverage ratio (GCR) သည် ဝင်ရိုးတစ်လုံးတည်းလှည့်သော (one axis tracking) စနစ်များအတွက်သာ သက်ရောက်သည်။ ၎င်းသည် array မှသိမ်းပိုက်ထားသော ခေါင်မိုး သို့မဟုတ် မြေပြင်ဧရိယာနှင့် စုစုပေါင်း module ဧရိယာ၏အချိုးကို တိုင်းတာခြင်းဖြစ်သည်။ GCR 0.5 ဆိုသည်မှာ အလျားလိုက်ရှိသော ခေါင်မိုး သို့မဟုတ် မြေပြင်မျက်နှာပြင်အတွက် tracker ကို ၎င်းတို့အလျားလိုက်ဖြစ်နေစေရန် လှည့်ထားသောအခါ စုစုပေါင်းဧရိယာ၏ ထက်ဝက်ကို module များဖြင့် ဖုံးလွှမ်းထားသည်ဟုဆိုလိုသည်။ GCR နိမ့်လေ အတန်းကြားအကွာအဝေးပိုကျယ်လေ၊ မြင့်လေ အတန်းများကို ပိုမိုနီးကပ်စွာနေရာချထားလေဖြစ်သည်။ GCR 1 သည် ကပ်လျက်အတန်းများကြား နေရာလွတ်မရှိခြင်းကိုညွှန်ပြပြီး GCR 0 ဆိုသည်မှာ အတန်းများကြား အဓိပ္ပါယ်အားဖြင့် အကန့်အသတ်မဲ့အကွာအဝေးရှိခြင်းကိုဆိုလိုသည်။ ပုံမှန် one axis tracker စနစ်များသည် GCR သည် 0.3 မှ 0.6 အတွင်းရှိပြီး PVWatts တွင် ပုံသေသတ်မှတ်တန်ဖိုးမှာ 0.4 ဖြစ်သည်။ PVWatts ၏ ယခင်ဗားရှင်းများသည် GCR ကို သုညဟု ထိရောက်စွာယူဆခဲ့ပြီး ၎င်းမှာ လက်တွေ့တွင် ရရှိရန်မဖြစ်နိုင်ပါ။

## ၄ နေရောင်ခြည်စွမ်းအင်အရင်းအမြစ် (Solar Resource)

PVWatts သည် တစ်နှစ်တာအတွက် နာရီအလိုက်ဒေတာ လိုအပ်ပြီး ယင်းတွင် နေရောင်ခြည် irradiance ၏ အစိတ်အပိုင်းနှစ်ခု (beam နှင့် diffuse)၊ ပတ်ဝန်းကျင်အပူချိန် (dry bulb temperature) နှင့် မြေပြင်မှ 10m အမြင့်ရှိ လေတိုက်နှုန်းတို့ ပါဝင်သည်။ နာရီတိုင်းတွင် ဒေတာလိုင်းနှင့် သက်ဆိုင်သော ခုနှစ်၊ လ၊ ရက်နှင့် နာရီတို့ဖြင့် အချိန်တံဆိပ်တပ်ထားရမည် ဖြစ်ပြီး နေ၏တည်နေရာကို တိကျစွာ တွက်ချက်နိုင်စေရန်ဖြစ်သည်။

ရက်ထပ်နှစ်များ သို့မဟုတ် နေ့အလင်းရောင်ထိန်းသိမ်းသည့်အချိန် (daylight savings time) အတွက် ချိန်ညှိမှုများ မပြုလုပ်ပါ။ ထို့ကြောင့် ဤနေရာတွင် "နာရီအလိုက်ဒေတာ" ဟူသည်မှာ တစ်နှစ်အတွက် ဒေတာအချက် ၈၇၆၀ ကိုဆိုလိုသည်။ လိုအပ်သော အချက်အလက်များကို ဇယား ၄ နှင့် ၅ တွင် ဖော်ပြထားသည်။

## ၅ နေတည်နေရာ (Sun Position)

နာရီတစ်ခုစီတွင် PVWatts သည် [6] တွင်ဖော်ပြထားသော algorithm ကိုအသုံးပြု၍ နေတည်နေရာကို တွက်ချက်သည်။ နေတည်နေရာကို ထိုနာရီ၏ အလယ်မှတ်တွင် တွက်ချက်သည်- ဥပမာ၊ ညနေ ၂ နာရီမှ ၃ နာရီအတွက်၊ zenith နှင့် azimuth ထောင့်များကို ဆုံးဖြတ်ရန် ညနေ ၂:၃၀ တွင် တွက်ချက်သည်။ ၎င်းသည် နေမိုးကုပ်စက်ဝိုင်းအထက်တွင် တစ်နာရီလုံးရှိနေသော ပုံမှန်နေ့ခင်းနာရီများအတွက် ဖြစ်ရပ်ဖြစ်သည်။

နေထွက်ချိန် နာရီအတွက်၊ နေထွက်ချိန်နှင့် အချိန်အဆင့်၏ အဆုံးကြားရှိ အလယ်မှတ်ကို နေတည်နေရာတွက်ချက်ရန် အသုံးပြုသည်။ အလားတူပင် နေဝင်ချိန်နာရီအတွက် အချိန်အဆင့်၏အစနှင့် နေဝင်ချိန်ကြားရှိ အလယ်မှတ်ကို အသုံးပြုသည်။

## ၆ လှည့်ပတ်ခြင်း (Tracking)

PVWatts သည် ပုံသေ၊ ဝင်ရိုးတစ်လုံးတည်း သို့မဟုတ် ဝင်ရိုးနှစ်လုံးလှည့်သော (tracking) စနစ်များအတွက် ရောင်ခြည်ကျရောက်ထောင့် (Angle of Incidence - AOI) (α) တွက်ချက်မှုများ ပြုလုပ်သည်။

ပုံသေစနစ်များအတွက် မျက်နှာပြင်စောင်းထောင့် β၊ မျက်နှာပြင် azimuth γ၊ နေ၏ azimuth γ<sub>sun</sub> နှင့် နေ၏ zenith ထောင့် θ<sub>sun</sub> တို့ပေး၍ ရောင်ခြည်ကျရောက်ထောင့်အတွက် စံဂျီဩမေတြီတွက်ချက်မှုများ အကောင်အထည်ဖော်သည် (ညီမျှခြင်း ၁)။

α<sub>fixed</sub> = cos<sup>-1</sup>[sin(θ<sub>sun</sub>) cos(γ − γ<sub>sun</sub>) sin(β) + cos(θ<sub>sun</sub>) cos(β)] (1)

ဝင်ရိုးတစ်လုံးတည်း tracker များအတွက်၊ [10] တွင်မှတ်တမ်းတင်ထားသော algorithm ကိုအသုံးပြုသည်။ ၎င်းသည် စံပြ tracking ဟုယူဆပြီး မည်သည့်အရိပ်ကျခြင်းကိုမျှ ထည့်သွင်းမတွက်ချက်ပါ။ ဝင်ရိုးတစ်လုံးတည်း tracking algorithm သည် အလျားလိုက်မှ ±45 ဒီဂရီ လှည့်နိုင်မှုကန့်သတ်ချက်ကို code တွင်တိုက်ရိုက်ထည့်သွင်းထားသည်။ PVWatts သည် ground coverage ratio (GCR) ကိုအခြေခံ၍ ကပ်လျက်အတန်းများမှ အရိပ်ကျသည့် အတန်း၏အပိုင်းအစကို တွက်ချက်ရန် သီးခြား algorithm ကိုအသုံးပြုပြီး beam နှင့် diffuse irradiance များကို လျှော့ချသည်။ Backtracking ရွေးချယ်မှုကို ဖွင့်ထားသောအခါ PVWatts သည် ကပ်လျက်အတန်းများ၏ အလိုအလျောက်အရိပ်ကျခြင်းကို ရှောင်ရှားရန် SAM backtracking algorithm ကိုအသုံးပြုသည်။ ဤ algorithms များအကြောင်း အသေးစိတ်အတွက် [25] ကို ကြည့်ပါ။

**PVWatts V1 မှ ပြောင်းလဲမှု:** PVWatts V1 တွင် ဝင်ရိုးတစ်လုံးတည်း tracking သည် အလိုအလျောက်အရိပ်ကျခြင်း သို့မဟုတ် backtracking တစ်စုံတစ်ရာမရှိဟု ယူဆခဲ့သည်။ ၎င်းသည် တပ်ဆင်ထားသောစနစ်အများစုအတွက် လက်တွေ့မကျသောယူဆချက်ဖြစ်ပြီး tracking အတန်းများကို အကန့်အသတ်မဲ့အကွာအဝေးထား၍မရနိုင်ပါ။ ထို့ကြောင့် ဝင်ရိုးတစ်လုံးတည်းလှည့်သော စနစ်များအတွက် ထုတ်လုပ်မှုခန့်မှန်းချက်များသည် PVWatts V1 နှင့်ယှဉ်လျှင် လျှော့ချမည်ဖြစ်သည်။

ဝင်ရိုးနှစ်လုံး tracking စနစ်များအတွက် PV မျက်နှာပြင် စောင်းထောင့်နှင့် azimuth ကို နေ၏ zenith ထောင့်နှင့် azimuth ထောင့်တို့နှင့် အသီးသီးညီမျှအောင် သတ်မှတ်ပြီး ရောင်ခြည်ကျရောက်ထောင့်မှာ သုညဖြစ်သည်။ ဝင်ရိုးနှစ်လုံး tracking algorithm သည် အရိပ်ကျခြင်းမရှိဟု ယူဆသည်။

## ၇ Plane-of-Array Irradiance (POA)

Plane-of-array (POA) beam, ကောင်းကင်ပြန့်ကျဲရောင်ခြည်, နှင့် မြေပြင်မှအလင်းပြန်၍ပြန့်ကျဲရောင်ခြည် irradiance အစိတ်အပိုင်းများကို Perez 1990 algorithm [7] ကိုအသုံးပြု၍ တွက်ချက်သည်။ POA beam အစိတ်အပိုင်း I<sub>b</sub> သည် beam normal input ကို ရောင်ခြည်ကျရောက်ထောင့်၏ cosine နှင့်မြှောက်ခြင်းသာဖြစ်သည်။ သတ်မှတ်ထားသော empirical function F1 နှင့် F2 တို့မှ beam နှင့် diffuse input ပေး၍ isotropic, circumsolar, နှင့် horizon brightening diffuse terms များကို တွက်ချက်ပြီး မျက်နှာပြင်ပေါ်ရှိ စုစုပေါင်း sky diffuse I<sub>d,sky</sub> ကိုရရှိရန် ပေါင်းစပ်သည်။ စံ Perez မော်ဒယ်မှ အနည်းငယ် ပြုပြင်ထားမှုမှာ zenith ထောင့် ၈၇.၅ မှ ၉၀ ဒီဂရီကြားအတွက် diffuse irradiance ကို isotropic အဖြစ်သဘောထားခြင်းဖြစ်သည်။ မြေပြင်မှအလင်းပြန် irradiance I<sub>d,ground</sub> ကို စောင်းနေသောမျက်နှာပြင်နှင့်စပ်လျဉ်း၍ မြေပြင်မှ တွက်ချက်ထားသော view factor ဖြင့် isotropic diffuse အဖြစ် သဘောထားသည်။ Module cover ပေါ်သို့ ကျရောက်သော စုစုပေါင်း POA သည် အစိတ်အပိုင်း သုံးခု၏ ပေါင်းလဒ်ဖြစ်သည် (ညီမျှခြင်း ၂)။

I<sub>poa</sub> = I<sub>b</sub> + I<sub>d,sky</sub> + I<sub>d,ground</sub> (2)

Albedo (မြေပြင်မှအလင်းပြန်နှုန်း) ကို ပုံမှန်အားဖြင့် 0.2 ဟုပုံသေသတ်မှတ်ထားသည်။ TMY3 ဒေတာကို input အဖြစ်အသုံးပြုပြီး albedo ကော်လံတွင် တရားဝင်ဒေတာတွေ့ရှိပါက ဒေတာဖိုင်မှ နာရီအလိုက် albedo ကိုအသုံးပြုသည်။

## ၈ Module Cover

Module cover ပေါ်သို့ကျရောက်သော စုစုပေါင်း POA irradiance ပေး၍ PVWatts သည် အလင်းပြန်မှုကြောင့် ဆုံးရှုံးမှုများကိုတွက်ချက်ရန် တိုက်ရိုက် beam irradiance ကို AOI ပြုပြင်ချက်တစ်ခု ပြုလုပ်သည်။ ဤပြုပြင်ချက်သည် [19] တွင်အသုံးပြုသော module cover မှတဆင့် အလင်းဖြတ်သန်းနှုန်း (transmittance) ၏ ရူပဗေဒမော်ဒယ်၏ ပြုပြင်ထားသောဗားရှင်းကို အသုံးပြုသည်။

PVWatts V5 သည် အသုံးပြုသူများအား "Standard", "Premium", သို့မဟုတ် "Thin film" module များ၏ ရွေးချယ်ခွင့်ပေးထားသည်။ Standard နှင့် thin film module များအတွက် PVWatts သည် single slab ကိုအသုံးပြုပြီး 1.526 ရောင်ခြည်ပြန်ညွှန်းကိန်း (index of refraction) ရှိသောဖန်သားမှတဆင့် transmittance ကို တွက်ချက်သည်။ ၎င်းသည် [19] တွင် ကုသမှုကိုလိုက်နာပြီး အကျိုးသက်ရောက်မှုမရှိနိုင်ဟုဆုံးဖြတ်ထားသော absorptance term ကိုဖယ်ရှားခြင်းဖြင့် ရိုးရှင်းအောင်ပြုလုပ်ထားသည်။

Premium module ရွေးချယ်မှုအတွက် two slab နည်းလမ်းကိုအသုံးပြုပြီး ဖန်သားနှင့် angular response ကိုတိုးတက်ကောင်းမွန်စေရန် ဒီဇိုင်းထုတ်ထားသော ပါးလွှာသည့် anti-reflective (AR) coating နှစ်ခုစလုံးကို ပုံစံတူတွက်ချက်ရန် အသုံးပြုသည်။ Two slab မော်ဒယ်တွင် အရာဝတ္ထုနှစ်ခုမှတဆင့် irradiance ၏ transmittance ကို ခန့်မှန်းခြင်းပါဝင်သည်။ မော်ဒယ်သည် [19] တွင်ဖော်ပြထားသော မပြန့်ပွားသောရောင်ခြည်အတွက် ရူပဗေဒဆိုင်ရာဖော်ပြချက်ကို နှစ်ကြိမ်သက်ရောက်စေသည်- တစ်ကြိမ်သည် anti-reflective (AR) coating အတွက်ဖြစ်ပြီး နောက်တစ်ကြိမ်မှာ ဖန်သား cover အတွက်ဖြစ်သည်။

ပထမဦးစွာ AR coating ထဲသို့ အလင်းယိုင်ထောင့် θ<sub>2</sub> ကို ရောင်ခြည်ကျရောက်ထောင့် θ<sub>1</sub> ပေး၍ Snell's law ဖြင့်တွက်ချက်သည်:

θ<sub>2</sub> = arcsin ( (n<sub>air</sub>/n<sub>AR</sub>) sin(θ<sub>1</sub>) ) (3)

ထို့နောက် AR coating မှတဆင့် transmittance ကို Fresnel's equation မှတဆင့် အလင်းပြန်မှုမရှိသော မပြန့်ပွားသောရောင်ခြည်အတွက် တွက်ချက်သည်၊ ၎င်းသည် parallel နှင့် perpendicular အစိတ်အပိုင်းများ၏ ပျမ်းမျှကိုယူသည်။

τ<sub>AR</sub> = 1 - 0.5 [ (sin(θ<sub>2</sub> - θ<sub>1</sub>)² / sin(θ<sub>2</sub> + θ<sub>1</sub>)²) + (tan(θ<sub>2</sub> - θ<sub>1</sub>)² / tan(θ<sub>2</sub> + θ<sub>1</sub>)²) ] (4)

ဖန်သား cover ထဲသို့ အလင်းယိုင်ထောင့် θ<sub>3</sub> ကို Snell's law မှ ထပ်မံဆုံးဖြတ်သည်။

θ<sub>3</sub> = arcsin ( (n<sub>AR</sub>/n<sub>glass</sub>) sin(θ<sub>2</sub>) ) (5)

ဖန်သားမှတဆင့် transmittance ကို θ<sub>2</sub> နှင့် θ<sub>3</sub> ပေး၍ အလားတူတွက်ချက်သည်

τ<sub>glass</sub> = 1 - 0.5 [ (sin(θ<sub>3</sub> - θ<sub>2</sub>)² / sin(θ<sub>3</sub> + θ<sub>2</sub>)²) + (tan(θ<sub>3</sub> - θ<sub>2</sub>)² / tan(θ<sub>3</sub> + θ<sub>2</sub>)²) ] (6)

နောက်ဆုံးတွင် AR ကုတ်ထားသော module cover မှတဆင့် ထိရောက်သော transmittance ကို အောက်ပါအတိုင်းပေးသည်:

τ<sub>cover</sub> = τ<sub>AR</sub> * τ<sub>glass</sub> (7)

Desoto [19] သည် စုပ်ယူမှုကိုခန့်မှန်းရန် Bouguer's law ကိုသုံးရန်အကြံပြုသော်လည်း ပုံမှန်အတိုင်းအတာ (~2 mm) နှင့် extinction coefficient (K ≈ 4) အတွက် ဤအသုံးချမှုတွင် စုပ်ယူမှုသည် 0.1% ထက်နည်းမည်ဟု ခန့်မှန်းထားသောကြောင့် လျစ်လျူရှုထားသည်။

n<sub>air</sub> = 1 နှင့် n<sub>glass</sub> = 1.526 ပေး၍ AR coating အတွက် မတူညီသော ရောင်ခြည်ပြန်ညွှန်းကိန်းနှစ်ခုအတွက် angular response ကို ပုံ ၂ တွင် ပြသထားသည်။ မော်ဒယ်က AR coating သည် normal incidence တွင် standard glass ထက် transmittance 3% ခန့်တိုးတက်မှုရရှိသည်ဟု အကြံပြုပြီး ၎င်းသည် လုပ်ငန်းမှပြောဆိုချက်များ (Honeywell SOLARC [20], DSM Khepricoat [21]) နှင့် ကိုက်ညီပါသည်။

တုံ့ပြန်မှုမျဉ်းကွေးကို normal incidence တွင်ပုံမှန်ဖြစ်အောင် (normalize) ပြုလုပ်ခြင်းသည် AR glass အတွက် ကျရောက်ထောင့်မြင့်မားချိန်များတွင် အလင်းဖမ်းယူမှု တိုးတက်မှုကို မော်ဒယ်က ခန့်မှန်းကြောင်း ပြသပြီး ၎င်းသည် AR coated module များထုတ်လုပ်သူများ၏ဒေတာနှင့် ကိုက်ညီသည် (ပုံ ၃)။

**PVWatts V1 မှ ပြောင်းလဲမှု:** PVWatts V1 တွင် angular response correction ကို ကျရောက်ထောင့် ၅၀ ဒီဂရီထက်ကြီးမှသာ သက်ရောက်ခဲ့သည်။ နည်းလမ်းသစ်သည် ရူပဗေဒမော်ဒယ်ကိုအသုံးပြုသဖြင့် standard glass နှင့် anti-reflective glass အကြား ပုံမှန်ကွာခြားချက်များကို ပုံစံတူလုပ်ရန် ဖြစ်နိုင်သည်။

AR glass ရွေးချယ်မှုသည် တိုးတက်ကောင်းမွန်သော angular response ကြောင့် standard option နှင့်ယှဉ်လျှင် output အနည်းငယ်ပိုမိုမြင့်မားမည်ဟု ခန့်မှန်းသည်။ Module cover ၏ normal incidence transmittance ကို စနစ်၏ nameplate rating တွင် ဖမ်းယူထားသည်ဟုယူဆသည်။ AR glass အတွက် ပိုမိုကောင်းမွန်သော angular response ကြောင့် ထပ်ဆောင်းစွမ်းအင်ထွက်ရှိမှုမှာ အနည်းငယ်သာဖြစ်သည်- Texas ရှိ တောင်ဘက်မျက်နှာမူ ပုံသေစနစ်အတွက် anti-reflective coating သည် တစ်နှစ်အခြေခံအားဖြင့် စွမ်းအင် ∼0.5% ပိုမိုထွက်ရှိခဲ့သည်။ ၎င်းသည် တည်နေရာနှင့် စနစ် parameters များပေါ်မူတည်၍ ကွဲပြားလိမ့်မည်။

## ၉ အပူဆိုင်ရာမော်ဒယ် (Thermal Model)

PVWatts သည် Fuentes [15] မှတီထွင်သော ပထမနိယာမအပူလွှဲပြောင်းမှုစွမ်းအင်ချိန်ခွင်လျှာမော်ဒယ်ကိုအသုံးပြု၍ လည်ပတ်ဆဲလ်အပူချိန် T<sub>cell</sub> ကိုတွက်ချက်ရန် အပူဆိုင်ရာမော်ဒယ်ကို အကောင်အထည်ဖော်ထားသည်။ Fuentes မော်ဒယ်တွင် module ၏ အပူလှောင်နိုင်စွမ်း (thermal capacitance) ၏သက်ရောက်မှုများပါဝင်ပြီး အပူနောက်ကျခြင်း (thermal lag) transient behavior အတွက် အချိန်အဆင့်များကြား numerical integration ကိုလုပ်ဆောင်သည်။ အပူဆိုင်ရာမော်ဒယ်သည် လည်ပတ်ဆဲလ်အပူချိန်ကိုတွက်ချက်ရန် စုစုပေါင်း POA irradiance၊ လေတိုက်နှုန်းနှင့် အပူချိန်တို့ကိုအသုံးပြုသည်။ PVWatts သည် ရာသီဥတုဒေတာထဲမှ လေတိုက်နှုန်းကို ပြုပြင်ရာတွင် မြေပြင်မှ 5 m အမြင့်ရှိသည်ဟုယူဆပြီး module ၏ installed nominal operating cell temperature (INOCT) ကို 45°C ဟုယူဆသည်။

**PVWatts V1 မှ ပြောင်းလဲမှု:** PVWatts V5 တွင် ပုံသေစနစ်များကို open rack သို့မဟုတ် roof mount ပေါ်တွင်တပ်ဆင်နိုင်သည်။ ရွေးချယ်မှုသည် လေစီးဆင်းမှုလျော့နည်းပြီး roof mount စနစ်၏ ပိုမိုမြင့်မားသော လည်ပတ်အပူချိန်အပေါ် အခြေခံ၍ ယူဆထားသော INOCT ကိုပြောင်းလဲစေသည်။

Fuentes စာတမ်းတွင် mounting configuration ပေါ်အခြေခံ၍ စံအခြေအနေများ (800 W/m², 20°C ambient) တွင်တိုင်းတာသော nominal operating cell temperature (NOCT) ကို INOCT သို့ဘာသာပြန်ဆိုချက်ကို ဆွေးနွေးထားသည်။ Roof mount စနစ်အတွက် module များတဝိုက် လေစီးဆင်းမှုသည် open rack ထက် ပိုမိုကန့်သတ်ထားပြီး installed nominal operating temperature သည် ပိုမိုမြင့်မားလိမ့်မည်။ ခန့်မှန်းချက်အရ INOCT သည် 4 inch standoffs အတွက် အကြမ်းဖျင်း 49°C ဖြစ်သည်။ Fixed open rack နှင့် tracking စနစ်များအတွက် မူလ PVWatts V1 ၏ ယူဆချက်ဖြစ်သည့် 45°C INOCT ကို ဆက်လက်ထိန်းသိမ်းထားသည်။

## ၁၀ Module မော်ဒယ် (Module Model)

PVWatts module model သည် တွက်ချက်ထားသော ဆဲလ်အပူချိန် T<sub>cell</sub> နှင့် ဖြတ်သန်းလာသော POA irradiance I<sub>tr</sub> တို့ပေး၍ သတ်မှတ်ထားသော nameplate DC rating P<sub>dc0</sub> ရှိသည့် array မှ DC power ကိုတွက်ချက်သည်။ Array စွမ်းဆောင်ရည်သည် temperature coefficient γ ဖြင့်ထိန်းချုပ်ကာ အပူချိန်တိုးလာမှု၏ function တစ်ခုအနေဖြင့် linear နှုန်းဖြင့် ကျဆင်းသည်ဟု ယူဆသည်။ ရည်ညွှန်းဆဲလ်အပူချိန် T<sub>ref</sub> သည် 25°C ဖြစ်ပြီး ရည်ညွှန်း irradiance သည် 1000 W/m² ဖြစ်သည်။

P<sub>dc</sub> = (I<sub>tr</sub> / 1000) * P<sub>dc0</sub> * (1 + γ * (T<sub>cell</sub> - T<sub>ref</sub>)) (8)

**PVWatts V1 မှ ပြောင်းလဲမှု:** PVWatts V1 တွင် irradiance 125 W/m² ထက်နည်းသောအခြေအနေအတွက် output ကိုလျှော့ချရန် quadratic correction ကိုအသုံးပြုခဲ့သည်။ စနစ်များစွာ၏လည်ပတ်ဒေတာများနှင့်နှိုင်းယှဉ်ရာတွင် ဤအပြုအမူကို ခေတ်မီစနစ်များတွင် မတွေ့ရှိရသောကြောင့် correction ကိုဖယ်ရှားခဲ့သည်။

Temperature coefficient သည် ရွေးချယ်လိုက်သော module အမျိုးအစားပေါ် မူတည်သည်။ အသုံးပြုသောတန်ဖိုးများကို CEC module database ရှိ module ပေါင်း ၁၁၀၀၀ ကျော်ကို ကိန်းဂဏန်းဆိုင်ရာခွဲခြမ်းစိတ်ဖြာမှုမှ ဆုံးဖြတ်ထားပြီး ဇယား ၃ တွင်ဖော်ပြထားသည်။

**ဇယား ၃။ မတူညီသော module အမျိုးအစားများအတွက် ယူဆချက်များ (Assumptions for different module types)**

| Module အမျိုးအစား | စွမ်းဆောင်ရည် | Cover အမျိုးအစား | Temperature coefficient |
| :--- | :--- | :--- | :--- |
| Standard | ~15% | Glass (ဖန်သား) | -0.47% / °C |
| Premium | ~19% | Anti-reflective (အလင်းပြန်မှုဆန့်ကျင်) | -0.35% / °C |
| Thin film | ~10% | Glass (ဖန်သား) | -0.20% / °C |

**PVWatts V1 မှ ပြောင်းလဲမှု:** PVWatts V1 သည် ပုံသေ temperature coefficient -0.5%/°C ကိုအသုံးပြုခဲ့သည်။ တန်ဖိုးအသစ်များသည် အသစ်သော module များ၏ စွမ်းဆောင်ရည်မြှင့်တက်လာမှုကို ညွှန်ပြသည်။

## ၁၁ စနစ်ဆုံးရှုံးမှုများ (System Losses)

မော်ဒယ်တွင် အတိအလင်း ပုံစံမတူတွက်ချက်ထားသော စနစ်ဆုံးရှုံးမှုများကို DC စွမ်းအင်၏ ရာခိုင်နှုန်းတစ်ခုအဖြစ် အသုံးပြုသူမှ ပေးအပ်သည်။ ဤနံပါတ်ဖြင့် ကိုယ်စားပြုသော ဆုံးရှုံးမှုများတွင် soiling (ဖုန်ကပ်ခြင်း)၊ အရိပ်ကျခြင်း၊ ဆီးနှင်းဖုံးခြင်း၊ mismatch (မကိုက်ညီခြင်း)၊ ဝါယာကြိုးသွယ်တန်းခြင်း၊ ဆက်သွယ်မှုများ၊ light induced degradation (အလင်းကြောင့်ယိုယွင်းခြင်း)၊ nameplate rating၊ စနစ်သက်တမ်းနှင့် လည်ပတ်နိုင်မှုတို့၏ သက်ရောက်မှုများ ပါဝင်သည်။ ပုံသေသတ်မှတ်တန်ဖိုးများကို PVWatts V1 တွင်အသုံးပြုသော ဆက်စပ် derate များနှင့်အတူ ဇယား ၆ တွင် ဖော်ပြထားသည်။ V5 တွင် ဝါယာဆုံးရှုံးမှုအားလုံးကို ဆုံးရှုံးမှုအမျိုးအစားတစ်ခုထဲသို့ ပေါင်းစည်းထားပြီး inverter ကို အတိအလင်းပုံစံတူတွက်ချက်ထားသည်။ ဆုံးရှုံးမှုယူဆချက်များအတွက် [12] ကိုကြည့်ပါ။

| Inverter | - | 0.92 |

စုစုပေါင်းဆုံးရှုံးမှုသည် တစ်ခုချင်းစီဆုံးရှုံးမှု၏ ပေါင်းလဒ်မဟုတ်ကြောင်း သတိပြုရန်အရေးကြီးသည်။ စုစုပေါင်းဆုံးရှုံးမှုကို ညီမျှခြင်း ၉ တွင်ပြသထားသည့်အတိုင်း ဆုံးရှုံးမှုတစ်ခုစီ L<sub>i</sub> (%) မှ လျှော့ချမှုကို မြှောက်ခြင်းဖြင့် တွက်ချက်သည်။ ပုံသေသတ်မှတ်ထားသော စုစုပေါင်းစနစ်ဆုံးရှုံးမှုသည် 14% ဟုတွက်ချက်ထားသည်။

L<sub>total</sub>(%) = 100 [ 1 - ∏<sub>i</sub> (1 - (L<sub>i</sub> / 100)) ] (9)

အရိပ်ကျခြင်းဆုံးရှုံးမှုအတွက် ပုံသေသတ်မှတ်ယူဆချက်သည် အဆောက်အဦးကြီးများ၊ တောင်များ သို့မဟုတ် အခြားအတားအဆီးများကဲ့သို့ အဝေးမှ အရာဝတ္ထုများကြောင့် မိုးကုပ်စက်ဝိုင်းကို ပိတ်ဆို့ခြင်းကို ကိုယ်စားပြုသည်။ တပ်ဆင်ထားသောစနစ်များကို စစ်တမ်းကောက်ယူမှုများအရ "အရိပ်မရှိ" ဟုဖော်ပြထားသော စနစ်များတွင် အရိပ်ကျခြင်းကြောင့် ပျမ်းမျှဆုံးရှုံးမှုသည် အကြမ်းဖျင်း 3% ဖြစ်သည် [14]။ အနီးနားရှိ သစ်ပင်များ သို့မဟုတ် အဆောက်အဦများကြောင့် အရိပ်ကျနေသော စနစ်တစ်ခုအတွက် ဆုံးရှုံးမှုတန်ဖိုးကို ပြင်ပအရိပ်ခွဲခြမ်းစိတ်ဖြာခြင်း software ကိုသုံး၍ သို့မဟုတ် ဆိုက်တွင်းစစ်တမ်းဖြင့် အရိပ်ကျဆုံးရှုံးမှုကိုကြိုတင်ခန့်မှန်းတွက်ချက်ခြင်းဖြင့် သင့်လျော်စွာတိုးမြှင့်သင့်သည်။

Light-induced degradation (LID) ဆိုသည်မှာ module သည် နေရောင်ခြည်ကိုပထမဆုံးအကြိမ်ထိတွေ့သောအခါ ၎င်း၏ power output ကျဆင်းသွားသည့်ဖြစ်စဉ်ဖြစ်သည်။ ဤကနဦးကာလပြီးနောက် module power သည် တည်ငြိမ်သွားပြီး တပ်ဆင်သက်တမ်းတစ်လျှောက် ပုံမှန်ရေရှည်ယိုယွင်းမှုနှုန်း (∼0.5%/year) အတိုင်း ဆက်လက်ဖြစ်ပေါ်သည်။ ပုံသေသတ်မှတ်ထားသော LID ဆုံးရှုံးမှု 1.5% သည် မတူညီတဲ့ module အမျိုးအစားများတွင် ဆုံးရှုံးမှုတိုင်းတာချက်များကိုအခြေခံ၍ ပုံမှန်တန်ဖိုးဖြစ်သည် [13]။ အချို့ premium module များသည် ၎င်းတို့၏ပစ္စည်းများနှင့်ဖွဲ့စည်းပုံကြောင့် ပိုမိုနိမ့်သော LID ဆုံးရှုံးမှုကိုကြုံတွေ့ရနိုင်ပြီး အခြားသူများမှာ 1.5% ထက်ပိုမိုသော LID ဆုံးရှုံးမှုကိုကြုံတွေ့ရနိုင်သည်။

**PVWatts V1 မှ ပြောင်းလဲမှု:** PVWatts V1 သည် ပုံသေသတ်မှတ်တန်ဖိုး 0.77 ရှိသော DC-to-AC derate factor ကိုအသုံးပြုခဲ့သည်။ စနစ်ဆုံးရှုံးမှုရာခိုင်နှုန်း input အဖြစ်ပြောင်းရခြင်းမှာ PVWatts ကို လုပ်ငန်းခွင်တွင်အသုံးများသောအလေ့အထနှင့်ကိုက်ညီစေရန်၊ derate factor သဘောတရားနှင့်မရင်းနှီးသောသူများအတွက် inputs များကို ပိုမိုနားလည်လွယ်စေရန်ဖြစ်သည်။ Inverter စွမ်းဆောင်ရည်ကို စနစ်ဆုံးရှုံးမှုတွင်မပါဝင်စေဘဲ သီးခြား input parameter ဖြစ်သည်။ PVWatts V5 စနစ်ဆုံးရှုံးမှုကို PVWatts V1 DC-to-AC derate factor သို့ အကြမ်းဖျင်းပြောင်းလဲရန်- 1. စနစ်ဆုံးရှုံးမှုကို derate သို့ပြောင်းပါ- 1 - 14/100 = 0.86။ 2. ဤတန်ဖိုးကို nominal inverter စွမ်းဆောင်ရည်နှင့်မြှောက်ပါ- 0.86 × 0.96 = 0.825 ၎င်းက PVWatts V5 ၏ပုံသေစနစ်ဆုံးရှုံးမှုသည် အဆင့်မြှင့်ထားသော input ယူဆချက်များကြောင့်သာ PVWatts V1 ထက် စနစ်စွမ်းဆောင်ရည် အကြမ်းဖျင်း 7% တိုးတက်မှုကိုကိုယ်စားပြုကြောင်း ညွှန်ပြသည်။ ဗားရှင်း 5 တွင် ပြန်လည်ပြင်ဆင်ထားသော inverter စွမ်းဆောင်ရည်မျဉ်းကွေး၏အကျိုးသက်ရောက်မှု (နောက်အပိုင်းတွင်ဖော်ပြထားသည်) သည် V1 နှင့်ယှဉ်ပါက နှစ်စဉ်စွမ်းအင်အခြေခံပေါ်တွင် ရရှိသောစွမ်းဆောင်ရည်မှာ 8-9% နီးပါးရှိစေသည်။ ဤအပြုအမူသည် PVWatts အသုံးပြုသူများထံမှရာနှင့်ချီသောအစီရင်ခံစာများ၊ ကျွမ်းကျင်သူစစ်တမ်းကောက်ယူမှုနှင့် မြောက်မြားစွာသောတိုင်းတာရေးဒေတာအစုံများကို ချိန်ညှိခြင်းတို့နှင့်ကိုက်ညီပြီး ယခင် PVWatts derate သည် ရှေးရိုးဆန်လွန်းပြီး ခေတ်မီစနစ်စွမ်းဆောင်ရည်ကို ပျမ်းမျှအားဖြင့် အနည်းဆုံး 8-9% လျှော့တွက်ခဲ့ကြောင်း ညွှန်ပြသည်။

## ၁၂ Inverter မော်ဒယ် (Inverter Model)

PVWatts V5 ရှိ inverter မော်ဒယ်သည် ကယ်လီဖိုးနီးယားစွမ်းအင်ကော်မရှင် (CEC) inverter စွမ်းဆောင်ရည်ဒေတာကို ခွဲခြမ်းစိတ်ဖြာမှုအပေါ်အခြေခံသည်။ ၂၀၁၀ ခုနှစ်နောက်ပိုင်း inverter များကို ခွဲခြမ်းစိတ်ဖြာမှုတွင် ထည့်သွင်းခဲ့သည်။ "ပုံမှန်" inverter တစ်ခုကို ပျမ်းမျှ part-load စွမ်းဆောင်ရည်မျဉ်းကွေးကိုဦးစွာတွက်ချက်ပြီး ထို့နောက် အမှန်တကယ်စွမ်းဆောင်ရည်မျဉ်းကွေးသည် ပျမ်းမျှနှင့်အနီးစပ်ဆုံးဖြစ်သော dataset ထဲရှိ အမှန်တကယ် inverter ကိုရှာဖွေခြင်းဖြင့် ရွေးချယ်ခဲ့သည်။ ထိုစွမ်းဆောင်ရည်ဒေတာကို [22] တွင်တင်ပြထားသည့်အတိုင်း quadratic ဆုံးရှုံးမှုမော်ဒယ်သို့ fit လုပ်ခဲ့သည်။ ရလာသောစွမ်းဆောင်ရည်မျဉ်းကွေးကို ညီမျှခြင်း 10 တွင်ပေးထားပြီး ၎င်းသည် အသုံးပြုသူသတ်မှတ်သော nominal rated efficiency η<sub>nom</sub> သို့ စွမ်းဆောင်ရည်မျဉ်းကွေးကို ချိန်ညှိပေးသည်။ CEC ဒေတာမှ အမှန်တကယ် အများဆုံးပုံမှန် inverter အတွက် ရည်ညွှန်း inverter စွမ်းဆောင်ရည် η<sub>ref</sub> သည် 0.9637 (P<sub>ac0</sub>/P<sub>dc0</sub>) ဖြစ်ပြီး AC nameplate rating P<sub>ac0</sub> ကို စနစ်၏ DC rating နှင့် DC-to-AC ratio မှဆုံးဖြတ်သည်။ ပုံသေသတ်မှတ်ထားသော nominal efficiency η<sub>nom</sub> မှာ 0.96 ဖြစ်သည်။

η = (η<sub>nom</sub> / η<sub>ref</sub>) * ( -0.0162 * ζ - (0.0059 / ζ) + 0.9858 )
ζ = P<sub>dc</sub> / P<sub>dc0</sub>
P<sub>dc0</sub> = P<sub>ac0</sub> / η<sub>nom</sub> (10)

ခန့်မှန်းထားသော AC output သည် nameplate rating ထက်ကျော်လွန်သောအခါ output ကို nameplate တန်ဖိုးသို့ ကန့်သတ်ထားသည်။ Inverter စွမ်းဆောင်ရည်ကို မတူညီသော nominal efficiencies အတွက် ပုံ ၄ တွင်ပြသထားပြီး မူရင်း version 1 မျဉ်းကွေးနှင့်အတူ ဖော်ပြထားသည်။

P<sub>ac</sub> =
- η P<sub>dc</sub> : 0 < P<sub>dc</sub> < P<sub>dc0</sub>
- P<sub>ac0</sub> : P<sub>dc</sub> ≥ P<sub>dc0</sub>
- 0 : P<sub>dc</sub> = 0 (11)

## ၁၃ မော်ဒယ်၏ အထုတ်များ (Model Outputs)

PVWatts သည် စနစ်အသေးစိတ်နှင့် နာရီအလိုက် irradiance၊ အပူချိန်နှင့် လေတိုက်နှုန်းဒေတာများအပေါ်အခြေခံ၍ နာရီအလိုက်အထုတ်များစွာကို အစီရင်ခံသည်။ ၎င်းတို့ကို ဇယား ၇ တွင် အကျဉ်းချုပ်ဖော်ပြထားသည်။

**ဇယား ၇။ တွက်ချက်ထားသော နာရီအလိုက် အထုတ်များ (Hourly Calculated Outputs)**

| ကွက်လပ် | ယူနစ် |
| :--- | :--- |
| Incident POA irradiance (POA ကျရောက်သည့် irradiance) | W/m² |
| Transmitted POA irradiance (ဖြတ်သန်းသွားသော POA irradiance) | W/m² |
| DC power (DC ပါဝါ) | W |
| AC power (AC ပါဝါ) | W |

ထို့အပြင် လစဉ်တစ်ရက်ပျမ်းမျှ incident POA irradiance ကို အသုံးပြုသူထံ အစီရင်ခံပါသည်။ လ m အတွက် ပျမ်းမျှ POA (kW/m²/day) ကို ညီမျှခြင်း 12 ဖြင့်ပေးသည်။

POA<sub>m</sub> = (0.001 * Σ<sub>m</sub> POA<sub>h</sub>) / (လ m ၏ ရက်အရေအတွက်) (12)

နာရီအလိုက်အထုတ်များဖြစ်သော DC နှင့် AC power များကိုလည်း အသုံးပြုသူထံအစီရင်ခံသည့် လစဉ်နှင့်နှစ်စဉ်စွမ်းအင်စုစုပေါင်းများအဖြစ် စုစည်းတင်ပြသည်။

ဇယား ၈ တွင် အစီရင်ခံထားသော အခြေအနေတစ်ခုစီအတွက် နံပါတ်သည် PVWatts V5 နှင့် V1 နှစ်စဉ် AC စွမ်းအင်ထုတ်လုပ်မှုခန့်မှန်းချက်ကြား ရာခိုင်နှုန်းကွာခြားချက်ဖြစ်သည်။ ရလဒ်များက PVWatts V5 သည် စနစ်ဖွဲ့စည်းပုံအားလုံးနှင့် တည်နေရာအားလုံးတွင် နှစ်စဉ်စွမ်းအင်ပမာဏပျမ်းမျှ 8% ပိုမိုခန့်မှန်းသည်ကိုပြသသည်။ One axis tracking ကိစ္စများတွင် တိုးတက်မှုနည်းပါးသည် - ၎င်းမှာ PVWatts V1 က one axis tracking အတွက် အတန်းကြားအရိပ်မကျဟူသော ယူဆချက်ဖြင့် ခန့်မှန်းမှုလွန်ကဲခြင်းကြောင့်ဖြစ်သည်။ ပုံသေစနစ်များအတွက် ပိုမိုကြီးမားသော တိုးတက်မှုမှာ ၎င်းတို့သည် နေကိုလှည့်မချိန်သောကြောင့် part-load တွင် ပိုမိုမကြာခဏလည်ပတ်သောကြောင့်ဖြစ်ပြီး ထို့ကြောင့် ပိုမိုကောင်းမွန်သော part-load inverter စွမ်းဆောင်ရည်၏အကျိုးသက်ရောက်မှုမှာ ပို၍ထင်ရှားသည်။

## ၁၅ တိုင်းတာရေးဒေတာနှင့် နှိုင်းယှဉ်ခြင်း (Comparison with Measured Data)

ဤအပိုင်းတွင် PVWatts ၏ခန့်မှန်းချက်များကို စနစ်များစွာအတွက် ပြသထားသည်။ စဉ်းစားထားသော စနစ် ၉ ခုကို [23] တွင်အသေးစိတ်ဖော်ပြထားပြီး ပုံသေစောင်းစနစ် ၈ ခုနှင့် ဝင်ရိုးတစ်လုံးလှည့်စနစ်တစ်ခုတို့ ပါဝင်သည်။ PVWatts V1 သည် နှစ်စဉ် AC စွမ်းအင်ကိုနှိုင်းယှဉ်သောအခါ စနစ်စွမ်းဆောင်ရည်ကို 11.9% လျှော့တွက်ခဲ့ပြီး PVWatts V5 မှာ 1.8% သာနိမ့်ပါသည်။ စဉ်းစားထားသောစနစ်အားလုံးသည် အရိပ်မရှိကြဘဲ စနစ်မရရှိနိုင်သောကာလများကို နှိုင်းယှဉ်မှုမှဖယ်ရှားခဲ့သည်။ ထို့ကြောင့် မော်ဒယ်နှစ်ခုစလုံးအတွက် အရိပ်ကျခြင်းနှင့်လည်ပတ်နိုင်မှုဆုံးရှုံးမှုယန္တရားများကို သုညဟုသတ်မှတ်ခဲ့သည်။ စံမဟုတ်သော module အမျိုးအစားများရှိသည့်စနစ်များကို PVWatts V5 တွင် သင့်လျော်စွာဖွဲ့စည်းခဲ့သည်- စနစ် ၂ (premium)၊ စနစ် ၇ (thin-film) နှင့် စနစ် ၉ (thin-film)။

ပုံ ၆ တွင် Colorado ရှိ ပုံသေ crystalline silicon စနစ်အတွက် PVWatts V1 နာရီအလိုက်ရလဒ်များကို ပြသထားသည်။ ဒေတာသည် V1 မှ ခန့်မှန်းသည့် အနိမ့်ပါဝါအဆင့်များတွင် quadratic အပြုအမူကို မထောက်ခံပါ။ PVWatts V5 သည် 125 W/m² အောက် module စွမ်းဆောင်ရည်ညှိခြင်းကို မပါရှိသောကြောင့် တိုင်းတာရေးစနစ်ဒေတာနှင့် ပိုမိုကိုက်ညီသည်။ PVWatts V1 အနိမ့် irradiance ယူဆချက်များကို စဉ်းစားထားသော မည်သည့်ခေတ်မီစနစ်ကမျှ ထောက်ခံမှုမရှိပါ။ ထို့အပြင် V5 ရှိ အဆင့်မြှင့်ထားသော inverter စွမ်းဆောင်ရည်မျဉ်းကွေးသည် မြင့်မားသော power output များတွင် တိုင်းတာထားသောစနစ် output ကို ပိုမိုနီးကပ်စွာ ခြေရာကောက်သည်။

## ၁၆ အကျဉ်းချုပ် (Summary)

နာမည်ကြီး PVWatts photovoltaic စွမ်းဆောင်ရည်မော်ဒယ်အတွက် ဘက်စုံအဆင့်မြှင့်တင်မှုကို တင်ပြခဲ့သည်။ V5 တွင်တိုးတက်ကောင်းမွန်သော မော်ဒယ်ဖွဲ့စည်းပုံနှင့် အဆင့်မြှင့်ထားသော ပုံသေယူဆချက်များသည် အမှန်တကယ်စနစ်စွမ်းဆောင်ရည်နှင့်စပ်လျဉ်း၍ V1 ၏ ခန့်မှန်းမှုလျော့နည်းခြင်းကို အတိုင်းအတာတစ်ခုအထိ ပြုပြင်ပေးသည်။ အဆင့်မြှင့်ထားသော PVWatts ကို ၂၀၁၄ ခုနှစ် ဆောင်းဦးရာသီတွင် NREL PVWatts Online Calculator တွင်အသုံးပြုနိုင်ရန်ချထားမည်ဖြစ်ပြီး http://pvwatts.nrel.gov တွင်ရရှိနိုင်ပါမည်။
